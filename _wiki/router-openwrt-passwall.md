## OpenWRT PassWall 路由器代理配置教程

通过在 OpenWRT 路由器上配置 PassWall 插件，可以实现局域网内所有设备自动通过龙猫云机场加速，无需在每台电脑、手机、平板上单独安装客户端。这是家庭和办公室场景中最推荐的方案，一次配置，全家受益。

### 前提条件

1. **一台运行 OpenWRT 系统的路由器**（或刷了 OpenWRT 固件的路由器）。
2. **路由器已连接到互联网**，并能正常访问国内网站。
3. **已拥有龙猫云机场的有效订阅**和订阅链接。
4. **路由器的硬件配置**建议至少 256MB RAM 和 16MB Flash，推荐 512MB+ RAM 以获得更好的性能表现。

### 安装 PassWall 插件

如果您的 OpenWRT 系统尚未安装 PassWall 插件，请按照以下步骤安装：

1. 通过浏览器登录 OpenWRT 管理后台（默认地址为 192.168.1.1）。
2. 进入「系统」->「软件包」。
3. 点击「刷新列表」按钮，更新软件包索引。
4. 在「过滤器」输入框中输入 "luci-app-passwall"，点击「查找」。
5. 在搜索结果中找到 `luci-app-passwall`，点击右侧的「安装」按钮。
6. 安装完成后，在「服务」菜单中应该可以看到 PassWall 选项。

**备选安装方法**：如果在线安装失败，可以通过 SSH 登录路由器，使用以下命令安装：

```bash
opkg update
opkg install luci-app-passwall
```

如果您的路由器固件版本较旧，可能需要先添加软件源或更新系统。

### 配置 PassWall 插件

#### 基本设置

1. 进入「服务」->「PassWall」->「基本设置」。
2. 在「主开关」选项中，确保状态为「启用」。
3. 「运行模式」建议选择「自动切换模式」或「普通模式」。
4. 「TCP 协议」建议选择「V2Ray」或「Xray」，具体根据您的节点协议选择。

#### 添加订阅节点

1. 切换到「节点管理」->「订阅设置」选项卡。
2. 点击「添加」按钮，新增一个订阅配置。
3. 在「备注」栏输入「龙猫云机场」。
4. 在「订阅 URL」栏粘贴从龙猫云机场用户中心复制的订阅链接。
5. 「订阅类型」选择「V2Ray Subscription」或默认选项。
6. 点击「保存」按钮。
7. 回到「节点管理」页面，点击「更新订阅」按钮，等待节点列表加载完成。
8. 更新成功后，您可以在「节点管理」->「节点列表」中看到所有可用的节点。

#### 选择节点

1. 在「节点管理」->「节点列表」中，找到您想使用的节点。
2. 点击节点右侧的「TCP」按钮，将该节点设置为 TCP 代理节点。
3. 如果需要 UDP 代理（如游戏加速），也点击节点右侧的「UDP」按钮进行设置。
4. 返回「基本设置」页面，确认「主节点」已选择为您刚才设置的节点。

#### 启动代理

1. 确保所有设置已保存。
2. 在「基本设置」页面中，点击「启动」按钮启动 PassWall。
3. 稍等几秒钟，观察状态指示灯是否变为绿色，表示代理已成功运行。
4. 此时连接路由器的所有设备应该已经自动通过代理访问互联网。

### 高级配置

**自动切换**：PassWall 支持自动切换节点功能。当当前节点不可用时，自动切换到备用节点，确保网络连接不中断。

**访问控制**：您可以在「访问控制」中设置哪些设备走代理、哪些设备直连。例如，可以让您的电脑和手机走代理，而智能家居设备保持直连。

**DNS 设置**：建议在「高级设置」->「DNS」中开启 DNS 防泄漏，并设置加密 DNS 服务器（如 223.5.5.5 或 8.8.8.8）。

**分流规则**：PassWall 内置了丰富的分流规则，可以自动识别国内和海外流量，实现智能分流。您也可以在「规则管理」中自定义规则。

### 性能优化建议

1. **选择专用线路节点**：路由器的硬件性能有限，建议使用延迟较低、加密开销较小的节点（如 Shadowsocks 协议节点）。
2. **关闭不必要的功能**：如果路由器性能较低，可以关闭流量统计、日志记录等功能以减轻负载。
3. **定期更新订阅**：建议设置定时任务，定期自动更新订阅，确保节点列表始终最新。

### 常见问题

**Q：PassWall 启动后无法上网？**
A：请检查节点是否设置在 TCP 和 UDP 代理，以及节点是否有效。可以尝试在 PassWall 中切换不同的节点或协议。

**Q：路由器配置 PassWall 后速度变慢？**
A：路由器硬件性能有限，加密解密过程会消耗 CPU 资源。如果速度不理想，可以尝试更换性能更强的路由器，或选择加密开销较低的协议。

**Q：部分国内网站无法访问？**
A：PassWall 的默认分流策略可能导致某些国内网站流量走了代理。您可以在「访问控制」中添加这些网站到直连列表，或者在「规则管理」中更新规则集。
