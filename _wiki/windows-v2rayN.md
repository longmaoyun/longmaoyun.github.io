## Windows v2rayN 客户端配置教程

v2rayN 是 Windows 平台上最流行的代理客户端之一，功能强大、界面友好，支持 V2Ray、Shadowsocks、Trojan 等多种协议。本教程将详细介绍如何安装和配置 v2rayN 客户端，让您快速连接龙猫云机场的节点。

### 下载与安装

1. 访问龙猫云机场的 [下载页面](/download/)，找到 Windows 客户端下载区域。
2. 点击「下载 Windows 版」按钮，下载最新版本的 v2rayN 客户端压缩包。
3. 下载完成后，将压缩包解压到您选择的目录（建议不要放在系统盘 C 盘，避免权限问题）。
4. 进入解压后的文件夹，找到 `v2rayN.exe` 文件，双击运行即可启动客户端。

**注意**：首次运行时，Windows 可能会弹出安全警告，请点击「更多信息」->「仍要运行」。这是因为 v2rayN 是开源软件，未经过微软官方签名认证，但软件本身是安全可靠的。

### 导入订阅链接

1. 登录龙猫云机场官网，进入「用户中心」->「订阅管理」，复制您的专属订阅链接。
2. 在 v2rayN 主界面，点击顶部菜单栏的「订阅」->「订阅设置」。
3. 在弹出的订阅设置窗口中，点击「添加」按钮。
4. 在「备注」栏输入一个便于识别的名称（如「龙猫云机场」），在「URL」栏粘贴刚才复制的订阅链接，然后点击「确定」。
5. 回到主界面，再次点击「订阅」->「更新订阅」，客户端将自动从服务器获取所有节点信息。
6. 更新完成后，您可以在主界面的节点列表中看到所有可用的节点。

### 选择节点并连接

1. 在节点列表中，您可以看到每个节点的名称、协议类型、延迟等信息。
2. 右键点击您想要连接的节点，选择「设为活动服务器」。
3. 在系统托盘（屏幕右下角）找到 v2rayN 图标，右键点击。
4. 在弹出菜单中选择「系统代理」->「自动配置系统代理」或「全局模式」。
5. 此时系统代理已开启，您可以打开浏览器访问需要加速的网站进行测试。

### 代理模式说明

v2rayN 提供多种代理模式，适用于不同场景：

- **自动配置系统代理（PAC 模式）**：根据内置的 PAC 规则自动判断哪些网站走代理、哪些直连。这是最推荐的模式，既能加速海外网站，又不影响国内网站的访问速度。
- **全局模式**：所有网络流量都通过代理节点传输。适合需要全局代理的场景，但可能会影响国内网站的访问速度。
- **不改变系统代理**：不修改系统代理设置，需要手动在浏览器或其他应用中配置代理地址。
- **TUN 模式**：通过虚拟网卡实现系统级代理，可以代理所有应用的流量，包括不支持代理设置的应用。

### 高级功能

**节点测速**：在节点列表中，右键点击节点，选择「测试服务器真连接延迟」或「测试服务器速度」，可以查看节点的实际延迟和速度表现，帮助您选择最优节点。

**自动选择节点**：v2rayN 支持自动选择延迟最低的节点。在节点列表中，右键选择「自动配置系统代理」->「自动选择最低延迟」，客户端将自动切换到延迟最低的节点。

**自定义路由规则**：点击「设置」->「路由设置」，您可以自定义哪些域名或 IP 走代理、哪些直连，实现更精细的流量控制。

**开机自启**：点击「设置」->「参数设置」，勾选「开机启动」选项，v2rayN 将在系统启动时自动运行。

### 常见问题

**Q：v2rayN 提示「v2ray-core 未找到」怎么办？**
A：请确保 v2rayN 文件夹中包含 v2ray-core 的核心文件。如果缺失，请重新从官网下载完整包，或手动下载 v2ray-core 并放置到 v2rayN 目录下。

**Q：连接后无法访问国内网站？**
A：请将代理模式切换为「自动配置系统代理（PAC 模式）」，该模式会自动将国内网站流量直连，不影响国内访问。

**Q：更新订阅后节点列表没有变化？**
A：请检查订阅链接是否有效，尝试在浏览器中直接访问订阅链接，确认能正常返回数据。如果链接失效，请在用户中心重新生成订阅链接。

**Q：v2rayN 被 Windows Defender 报毒？**
A：这是误报，v2rayN 是开源软件，代码完全透明。您可以将 v2rayN 文件夹添加到 Windows Defender 的排除列表中，或暂时关闭实时保护后运行。
