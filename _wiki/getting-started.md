## 欢迎使用龙猫云机场

感谢您选择龙猫云机场！本指南将帮助您在 5 分钟内完成从注册到连接的全部流程，快速开启畅快的网络加速体验。

### 第一步：注册账号

1. 访问龙猫云机场官网首页（[www.longmaoyun.com](/)），点击页面右上角的「注册」按钮。
2. 在注册页面填写您的邮箱地址和设置密码。密码建议使用大小写字母+数字+特殊符号的组合，长度不少于 8 位，以确保账号安全。
3. 点击「获取验证码」，系统将发送一封验证邮件到您的注册邮箱。请登录邮箱查收验证码（如未收到，请检查垃圾邮件箱）。
4. 输入验证码后完成注册。注册成功后，系统会自动为您分配一个免费试用套餐，您可以立即开始体验我们的服务。

### 第二步：获取订阅链接

1. 使用注册的账号和密码登录官网。
2. 登录后进入「用户中心」，找到「订阅管理」页面。
3. 在订阅管理页面，您可以看到系统自动生成的订阅链接。点击「复制链接」按钮，将订阅链接复制到剪贴板。
4. **注意**：订阅链接是您使用龙猫云机场服务的关键凭证，请勿泄露给他人。如果链接不慎泄露，可以在订阅管理页面重置链接，旧链接将立即失效。

### 第三步：下载客户端

龙猫云机场支持全平台客户端，请根据您的设备选择对应的客户端下载：

- **Windows 用户**：推荐使用 v2rayN 或 Clash Verge 客户端
- **macOS 用户**：推荐使用 ClashX 或 Stash 客户端
- **Android 用户**：推荐使用 v2rayNG 或 Clash Meta 客户端
- **iOS 用户**：推荐使用 Shadowrocket（小火箭）或 Quantumult X
- **Linux 用户**：推荐使用命令行客户端或 GUI 客户端

所有客户端的下载链接和详细配置教程，请访问我们的 [下载页面](/download/) 和 [教程中心](/tutorial/)。

### 第四步：导入订阅并连接

以 Windows 平台的 v2rayN 客户端为例：

1. 下载并安装 v2rayN 客户端，启动程序。
2. 在系统托盘找到 v2rayN 图标，右键点击，选择「服务器」->「从剪贴板导入订阅链接」。
3. 或者打开 v2rayN 主界面，点击「订阅」->「订阅设置」，点击「添加」，粘贴订阅链接后点击「确定」。
4. 点击「订阅」->「更新订阅」，客户端将从服务器获取所有节点信息。
5. 在节点列表中双击任意一个节点，或者右键选择「设为活动服务器」。
6. 右键系统托盘图标，选择「系统代理」->「全局模式」或「自动配置系统代理」。
7. 打开浏览器，访问一个需要加速的网站，如果能够正常访问，说明连接成功！

### 第五步：验证连接

连接成功后，您可以通过以下方式验证：

1. 访问 [ip.sb](https://ip.sb) 或 [whatismyip.com](https://whatismyip.com)，检查 IP 地址是否已变为节点所在地区的 IP。
2. 访问 [google.com](https://google.com)、[youtube.com](https://youtube.com) 等网站，确认可以正常访问。
3. 在客户端中查看节点的延迟和流量使用情况，确认数据传输正常。

### 常见问题

**Q：订阅链接导入后没有节点显示怎么办？**
A：请在客户端中手动点击「更新订阅」按钮，确保客户端从服务器获取了最新的节点列表。如果仍然为空，请检查订阅链接是否有效。

**Q：连接后无法访问某些网站怎么办？**
A：尝试切换代理模式（如从「全局模式」切换到「PAC 模式」或「自动模式」），或者更换其他节点进行连接。

**Q：连接速度不理想怎么办？**
A：可以尝试切换到其他节点，或者选择延迟最低的节点。我们的日本和香港节点对中国大陆用户通常有较好的速度和稳定性。

**Q：如何确保连接安全？**
A：龙猫云机场所有节点均采用高强度加密协议传输数据，有效保护您的隐私安全。建议您始终通过官网获取订阅链接和服务信息，避免使用来源不明的第三方服务。
